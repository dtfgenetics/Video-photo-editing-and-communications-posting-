# Scope

## Product definition

Creator Engine is a local-first modular production engine for media processing, content packaging, and optional external communications/posting connectors.

## Locked rules

- Clean export is default.
- No forced watermark.
- No forced branding.
- No required social media API.
- No required paid API.
- Original media files are never modified.
- Generated files stay in runtime storage, not Git.

## MVP v0.1

MVP v0.1 is the Clean Posting Package Engine.

It must:

1. Import a video file.
2. Preserve the original.
3. Probe metadata.
4. Validate input.
5. Convert to clean MP4.
6. Export vertical, square, portrait, and landscape versions.
7. Generate thumbnail.
8. Create caption and hashtag placeholder files.
9. Create a package manifest.
10. Create a ZIP package.
11. Track job status.
12. Run through a recipe engine.

## Not in MVP v0.1

- Instagram auto-posting
- TikTok auto-posting
- Full video editor timeline
- Marketplace
- License/paywall system
- Team accounts
- Cloud sync
- Advanced analytics
- AI clip finder
